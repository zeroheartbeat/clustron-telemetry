# Clustron Telemetry

This repository stores anonymous usage telemetry for Clustron DKV.

The purpose is to understand:

• number of clusters using Clustron
• product versions in use
• feature adoption
• cluster sizes
• geographic distribution

## Privacy

Clustron does NOT collect:

• keys
• values
• user data
• credentials
• private IP addresses

Collected telemetry includes:

• product version
• cluster size
• enabled features
• operation counters
• country (derived from public IP)
• timestamp

Telemetry helps improve the product roadmap and prioritize development.

Telemetry can be disabled in Clustron configuration.
